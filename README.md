# Ecommerce REST API

## Endpoints
- GET    /api/ecommerce        — get all
- GET    /api/ecommerce/{id} — get by id
- POST   /api/ecommerce        — create
- PUT    /api/ecommerce/{id} — update
- DELETE /api/ecommerce/{id} — delete
- GET    /health                  — health check
- GET    /swagger                 — API docs
